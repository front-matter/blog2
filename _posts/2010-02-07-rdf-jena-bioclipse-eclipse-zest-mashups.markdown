Quite a while a go, I blogged about [Zest](http://www.eclipse.org/gef/zest/) in [Bioclipse](http://www.bioclipse.net/) showing
[a bit of ONS Solubility data <i class="fa-solid fa-recycle fa-xs"></i>](https://chem-bla-ics.linkedchemistry.info/2008/11/18/solubility-data-in-bioclipse-1.html). I could not follow up
on that until now, as I had yet to do a lot of [RDF](http://en.wikipedia.org/wiki/Resource_Description_Framework) work in
Bioclipse, so the screenshot back then was kind of a [mockup](http://en.wikipedia.org/wiki/Mockup).

Things are different now, and the Bioclipse-RDF functionality (using [Jena](http://jena.sourceforge.net/)) is released in
[Bioclipse 2.2](http://bioclipse.blogspot.com/2010/01/bioclipse-22-released.html) (see
[Semantic Web features in Bioclipse 2.2](http://chem-bla-ics.blogspot.com/2010/01/semantic-web-features-in-bioclipse-22.html)),
and I got around to writing the graphical goodies for the following papers. Not submitted yet, but here's the screenshot
showing a N3 file opened with a Zest-powered editor (read-only) and a plain text editor:

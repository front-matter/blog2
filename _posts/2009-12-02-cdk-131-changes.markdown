Two weeks ago, I released [CDK 1.2.4](http://chem-bla-ics.blogspot.com/2009/11/cdk-124-changes.html). [Anay](https://sourceforge.net/users/anaytamhankar/)
reported fails with generating the JavaDoc from the packages, which I think I both fixed now; the uploaded 1.2.4.1 packages on SourceForge include these
fixes.

The 1.2.4 release was soon followed by 1.3.1. Unfortunately, uploading the packages to to SourceForge over 3G with
[Chrome](http://www.google.com/chrome) did not work well, so only finished that today. CDK 1.3.1 is the second release
in the development branch, and brings in new functionality but also API changes. Here are the changes since the 1.3.0 release:
